Demo de Comunicación Segura

Esta demo muestra cómo un mensaje se cifra con AES-GCM, se distribuye la clave mediante Shamir Secret Sharing y el servidor solo recibe datos cifrados junto con referencias a custodios externos. El servidor devuelve la frase “Copia desde servidor: …” como prueba de proceso.

Para recuperar el contenido necesitarás reunir un número de shares igual al umbral configurado (THRESHOLD).
